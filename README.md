# ABEMA Player Plus

ABEMA の動画プレイヤーのワイド表示や擬似全画面を補助する軽量な Chrome 拡張機能です。

動画ページの移動、次エピソードへの遷移、プレイヤーの再描画などでワイド表示やフルスクリーン表示が解除される場面を補助します。

単純に自分がほしくて作りました。ブラウザで動画を垂れ流して見てる人におすすめです。

## Features

- ABEMA の対象動画ページでワイド表示や全画面を自動適用
- SPA 遷移や URL 変化後の再適用
- ワイド表示状態の矛盾を検出して再試行
- 表示モード切り替え
  - ワイド
  - ワイドシアター
- ABEMA プレイヤーの全画面ボタンを差し替える擬似全画面モード
- 最小限の権限構成
  - 使用権限は `storage` と `windows` のみ
  - 外部サーバーへの通信なし

## Target Pages

この拡張は `https://*.abema.tv/*` に content script を注入し、実際の処理は以下のページで行います。

- `/video/episode/` を含むページ
- `/channels/` を含むページ

## Install

Chrome Web Store 未公開の状態では、以下の手順で読み込めます。

1. [ZIPをダウンロード](https://github.com/shizuki7/_bemTv_ute_wide_mode/archive/refs/heads/main.zip) して展開する。
2. Chrome で `chrome://extensions/` を開く。
3. 右上の「デベロッパー モード」を有効にする。
4. 「パッケージ化されていない拡張機能を読み込む」をクリックする。
5. このプロジェクトのディレクトリを選択する。

## Usage

1. ABEMA の対象動画ページを開く。
2. 拡張機能の popup を開く。
3. 「有効」を ON にする。
4. 表示モードを選択する。

### ワイド

ABEMA のワイド表示ボタンを自動でクリックします。

### ワイドシアター

ワイド表示に加えて、サイドナビゲーション、ヘッダー、パンくずなどを抑制し、動画の表示領域を広げます。

### 擬似全画面

Popup の「擬似全画面ボタン」を ON にすると、ABEMA プレイヤーの全画面ボタンを差し替えます。差し替え後のボタンを押すと、Chrome ウィンドウのフルスクリーン化とプレイヤー周辺の疑似全画面 CSS を同時に実行します。

Google Chrome 以外の Chromium 系ブラウザでは、ウィンドウ内全画面になる場合があります。必要に応じてブラウザ側で手動フルスクリーンにしてください。

擬似全画面中は差し替えボタンがデフォルト表示へ戻るボタンとして動作します。ON の間は `f` キーによる ABEMA 標準フルスクリーンを抑制し、代わりに擬似全画面を切り替えます。

擬似全画面中は ABEMA のワイド表示 / デフォルト表示ボタンを非表示にし、解除時に通常表示へ戻します。

VOD とチャンネルページでプレイヤー DOM が異なるため、差し替えボタンの近くにあるプレイヤー wrapper を疑似全画面 root として固定します。

## Permissions

この拡張が要求する権限は以下のみです。

```json
["storage", "windows"]
```

`chrome.storage.local` は、以下の設定保存に使用します。

- `enabled`: 自動ワイド化の有効/無効
- `mode`: 表示モード
- `pseudoFullscreenEnabled`: ABEMA プレイヤーの全画面ボタン差し替えの有効/無効

`chrome.windows` は、差し替え後の ABEMA プレイヤー全画面ボタンから、現在の Chrome ウィンドウをフルスクリーン状態へ切り替えるために使用します。

この拡張は以下を行いません。

- 外部サーバーへの通信
- Cookie の読み取り
- タブ一覧の取得
- 閲覧履歴の取得
- ページ内入力値の収集
- 外部スクリプトの読み込み
- `eval` の使用

## Known Limitations

- ABEMA 側の DOM クラス名や SVG ファイル名が変わると、ボタン検出に失敗する可能性があります。
- ワイドシアターモードは ABEMA の現行 DOM/CSS 構造に依存します。
- SPA 遷移は `popstate`、タイトル変更、1秒間隔の URL ポーリングで検知します。
- Picture-in-Picture 操作は含みません。
- 広告、ライブ配信、特殊なプレイヤー UI では想定通り動作しない可能性があります。

## Note

この拡張は、ユーザーのブラウザ上で ABEMA のプレイヤー周辺レイアウトを調整する非公式ツールです。ABEMA の動画データ、配信内容、広告配信、DRM、課金機能を変更・回避・保存するものではありません。

ABEMA 公式の機能ではありません。利用する場合は、ABEMA の利用規約やサービス仕様を確認したうえで、自己責任で使用してください。

## Maintenance

この拡張は ABEMA のプレイヤー DOM や CSS クラスに依存しています。そのため、ABEMA 側の仕様やUI構造が変わった場合は、この拡張側でも対応が必要になる可能性があります。

ABEMA を利用している間は、自分の視聴環境で必要な範囲でメンテナンスしていく予定です。

需要がありそうであれば、将来的に Chrome Web Store での公開も検討します。
